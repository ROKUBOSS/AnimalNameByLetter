Format Specifiers in C

Data Type Format Specifier Description
Integer %d Prints an integer in decimal format.
Unsigned Integer %u Prints an unsigned integer.
Floating Point %f Prints a floating-point number.
Scientific Notation %e Prints a floating-point number in scientific notation.
Character %c Prints a single character.
String %s Prints a string of characters.
Octal Integer %o Prints an integer in octal format.
Hexadecimal Integer %x / %X Prints an integer in hexadecimal format. Use %X for uppercase letters.
Pointer %p Prints the address of a pointer.
Long Integer %ld Prints a long integer.
Short Integer %hd Prints a short integer. 
     
    
     
      #include <stdio.h>
     
      int main() {
        printf("Hello World\n");
        printf("Number 10+5\n");
        printf("Number %d\n",5);
        printf("Number %d\n",5+5);
        printf("Number %d\n",5-3);
        
        printf("Number %d\n",5*5);
        printf("Number %d\n",5/5);
        
        printf("Number %f\n",5 + 2.5); //Float By Default 6 Digits Value.
        printf("Number %.2f\n",5 + 2.5); //Float By Custom.
        printf("Number %lf\n",7 + 2.5); // Double By Default 6 Digits Value. 
        printf("Number %.3lf\n",7 + 2.5); //Float By Custom.
        
        printf("Letter %c\n",M);     //Wrong Statement
        printf("Letter %c\n", 'M');  //Right Statement

Types of Operators in C

      

          
              

                  Category
                  Symbol
                  Description
                  Example
              

          
          
              

                  Arithmetic Operators
                  + , - , * , / , %
                  Used for basic mathematical operations: addition, subtraction, multiplication, division, and modulus.
                  a + b
              

              

                  Relational Operators
                  == , != , > , < , >= , <=
                  Used to compare two values. Returns true (1) or false (0).
                  a > b
              

              

                  Logical Operators
                  && , || , !
                  Used for logical operations combining conditions.
                  a && b
              

              

                  Bitwise Operators
                  & , | , ^ , ~ , << , >>
                  Operate on bits of integer values.
                  a & b
              

              

                  Assignment Operators
                  = , += , -= , *= , /= , %=
                  Used to assign values to variables.
                  a = b
              

              

                  Increment/Decrement Operators
                  ++ , --
                  Used to increase or decrease a variable by 1.
                  a++
              

              

                  Ternary Operator
                  ?:
                  A shorthand for if-else statements.
                  result = (a > b) ? a : b;
              

              

                  Sizeof Operator
                  sizeof
                  Returns the size (in bytes) of a variable or data type.
                  sizeof(int)
              

              

                  Pointer Operators
                  * , &
                  Used to work with pointers; & gets address, * dereferences.
                  int *p = &a;
              

              

                  Member Selection Operators
                  . , ->
                  Access members of structures or unions.
                  ptr->member
