Format Specifiers in C

Data Type Format Specifier Description
Integer %d Prints an integer in decimal format.
Unsigned Integer %u Prints an unsigned integer.
Floating Point %f Prints a floating-point number.
Scientific Notation %e Prints a floating-point number in scientific notation.
Character %c Prints a single character.
String %s Prints a string of characters.
Octal Integer %o Prints an integer in octal format.
Hexadecimal Integer %x / %X Prints an integer in hexadecimal format. Use %X for uppercase letters.
Pointer %p Prints the address of a pointer.
Long Integer %ld Prints a long integer.
Short Integer %hd Prints a short integer.

কেন C প্রোগ্রামিং-এ %% ব্যবহার হয় ?

C প্রোগ্রামিং-এ %% ব্যবহার
C প্রোগ্রামিং-এ, দুইটি শতাংশ চিহ্ন (%%) printf বিবৃতিতে একটি একক শতাংশ চিহ্ন (%) প্রদর্শন করার জন্য ব্যবহৃত হয়।
এটি অবশিষ্টাংশ অপারেটরের সাথে সম্পর্কিত নয়, বরং printf ফাংশনের একটি ফরম্যাটিং বৈশিষ্ট্য।

printf-এ %% ব্যবহার ব্যাখ্যা
যখন আপনি আউটপুটে একটি শতাংশ চিহ্ন প্রদর্শন করতে চান,
তখন আপনাকে এটি %% ব্যবহার করে স্কেপ করতে হয়। 
এটি printf-কে জানায় যে এটি একটি অক্ষর হিসেবে বিবেচিত হবে, ফরম্যাট স্পেসিফায়ারের পরিবর্তে।
