1.Break Statement Example:

The `break` statement is used to exit a loop immediately when a certain condition is met.

Example: Break in a For Loop


#include <stdio.h>

int main() {
    // Loop from 1 to 10
    for(int i = 1; i <= 10; i++) 
      {
        // If i is 5, break the loop
        if(i == 5) 
        {
            break;
        }
        printf("%d\n", i);
    }

    return 0;
}


Explanation:
- This loop prints numbers from 1 to 10.
- When `i` reaches 5, the `break` statement is executed, and the loop stops.
- So, the output will only include the numbers 1 to 4.

Output:

1
2
3
4


2. Continue Statement Example:

The `continue` statement skips the current iteration of the loop and moves to the next iteration.

Example: Continue in a For Loop

#include <stdio.h>

int main() {
    // Loop from 1 to 10
    for(int i = 1; i <= 10; i++) 
      {
        // If i is 5, skip this iteration
        if(i == 5) 
        {
            continue;
        }
        printf("%d\n", i);
    }

    return 0;
}


Explanation:
- This loop prints numbers from 1 to 10.
- When `i` equals 5, the `continue` statement is executed, and the loop skips printing `5` but continues with the next numbers.
- So, the number 5 is skipped in the output.

Output:

1
2
3
4
6
7
8
9
10

Summary:
- `break`: Exits the loop entirely.
- `continue`: Skips the current iteration (‡¶™‡ßÅ‡¶®‡¶∞‡¶æ‡¶¨‡ßÉ‡¶§‡ßç‡¶§‡¶ø)  and continues with the next one.

These examples demonstrate (‡¶™‡ßç‡¶∞‡¶¶‡¶∞‡ßç‡¶∂‡¶® ‡¶ï‡¶∞‡ßá) how you can control the flow of loops in C using the `break` and `continue` statements.
