#include <stdio.h>

int main() {
    int year;
    printf("Enter a year: ");
    scanf("%d", &year);

    
    if ((year % 4 == 0 && year % 100 != 0) || (year % 400 == 0)) 
{
        printf("%d is a leap year.\n", year);
    } 
else {
        printf("%d is not a leap year.\n", year);
    }

    return 0;
}


`((year % 4 == 0 && year % 100 != 0) || (year % 400 == 0))` এই শর্তটি পুরোপুরি ব্যাখ্যা করলে, এটি একটি বছর অধিবর্ষ কিনা তা যাচাই করার জন্য ব্যবহৃত হয়। চলুন শব্দ ধরে ধরে বাংলায় এর ব্যাখ্যা করি:

1. **`year % 4 == 0`**:
   - এখানে `year % 4` বলতে বোঝানো হয়েছে, বছরটিকে ৪ দিয়ে ভাগ করলে ভাগশেষ ০ হবে কি না।
   - `== 0` এর মানে, বছরটি ৪ দিয়ে নিখুঁতভাবে ভাগ করা যায়।
   - অর্থাৎ, কোনো বছর ৪ দ্বারা বিভাজ্য হলে এই অংশটি সত্য হবে।

2. **`&&`**:
   - এটি "এবং" বোঝায়।
   - এর মানে হলো, এর আগে ও পরে থাকা দুটি শর্তই যদি সত্য হয়, তবেই পুরো অংশটি সত্য হবে।

3. **`year % 100 != 0`**:
   - এখানে `year % 100` বলতে বোঝানো হয়েছে, বছরটিকে ১০০ দিয়ে ভাগ করলে ভাগশেষ ০ হবে কি না।
   - `!= 0` মানে, বছরটি ১০০ দিয়ে ভাগ করলে ভাগশেষ ০ হবে না, অর্থাৎ বছরটি ১০০ দ্বারা বিভাজ্য নয়।
   - এর মানে, যদি কোনো বছর ১০০ দ্বারা বিভাজ্য না হয়, তবে এই অংশটি সত্য হবে।

4. **`||`**:
   - এটি "অথবা" বোঝায়।
   - এর মানে হলো, এর আগে বা পরে থাকা শর্তগুলোর যেকোনো একটি সত্য হলেই পুরো অংশটি সত্য হবে।

5. **`year % 400 == 0`**:
   - এখানে `year % 400` বলতে বোঝানো হয়েছে, বছরটিকে ৪০০ দিয়ে ভাগ করলে ভাগশেষ ০ হবে কি না।
   - `== 0` এর মানে, বছরটি ৪০০ দ্বারা নিখুঁতভাবে বিভাজ্য।
   - অর্থাৎ, কোনো বছর ৪০০ দ্বারা বিভাজ্য হলে এই অংশটি সত্য হবে।

### পুরো শর্তটি কী বোঝাচ্ছে?
- `(year % 4 == 0 && year % 100 != 0)`: বছরটি যদি ৪ দ্বারা বিভাজ্য হয় **এবং** ১০০ দ্বারা বিভাজ্য না হয়, তাহলে এটি অধিবর্ষ।
- `||`: অথবা
- `(year % 400 == 0)`: যদি বছরটি ৪০০ দ্বারা নিখুঁতভাবে বিভাজ্য হয়, তাহলে এটি অধিবর্ষ হবে।

### উদাহরণ:
- **২০২৪**:
  - `2024 % 4 == 0` সত্য, এবং `2024 % 100 != 0` সত্য, তাই এটি অধিবর্ষ।
  
- **১৯০০**:
  - `1900 % 4 == 0` সত্য, কিন্তু `1900 % 100 != 0` মিথ্যা (কারণ ১৯০০ ১০০ দ্বারা বিভাজ্য), তাই এটি অধিবর্ষ নয়।
  
- **২০০০**:
  - `2000 % 4 == 0` সত্য এবং `2000 % 400 == 0` সত্য, তাই এটি অধিবর্ষ।
