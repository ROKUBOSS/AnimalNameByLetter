নিচে একটি সি প্রোগ্রাম দেওয়া হল যা আয়তক্ষেত্রের দৈর্ঘ্য ও প্রস্থ দিয়ে তার পরিধি গণনা করে:


#include <stdio.h>

int main() {
    // Declare variables
    float length, width, perimeter;

    // Assign values
    length = 300; // দৈর্ঘ্য
    width = 200;  // প্রস্থ

    // Calculate perimeter
    perimeter = 2 * (length + width);

    // Display the result
    printf("আয়তক্ষেত্রের পরিধি: %.2f ইউনিট\n", perimeter);

    return 0;
}
```

### প্রোগ্রামটি কীভাবে কাজ করে:
1. **ভ্যারিয়েবল ডেক্লেয়ারেশন**: দৈর্ঘ্য, প্রস্থ এবং পরিধির জন্য ভ্যারিয়েবল ঘোষণা করা হয়েছে।
2. **মান নির্ধারণ**: দৈর্ঘ্য 300 এবং প্রস্থ 200 নির্ধারণ করা হয়েছে।
3. **পরিধি গণনা**: পরিধি গণনা করা হয়েছে উপরের ফর্মুলা অনুযায়ী।
4. **ফলাফল প্রদর্শন**: পরিধি স্ক্রীনে দেখানো হয়েছে।

আপনি যদি এই প্রোগ্রামটি চালান, তাহলে আউটপুট হবে:

আয়তক্ষেত্রের পরিধি: 1000.00 ইউনিট
-------------------------------------------------------------------------------------
সি প্রোগ্রামিং-এ **float** ডেটা টাইপ ব্যবহৃত হয় যাতে ভাসমান দশমিক সংখ্যা (যেমন 300.5 বা 200.75) সঠিকভাবে সংরক্ষণ এবং গণনা করা যায়। 

### কেন float ব্যবহার করা হয়:
1. **দশমিক মান**: যদি আপনার দৈর্ঘ্য এবং প্রস্থ দশমিক মানে হয় (যেমন 300.5, 200.75), তাহলে float ব্যবহার করা দরকার।
2. **গণনার সঠিকতা**: কিছু গণনায় দশমিক সংখ্যা থাকতে পারে, তাই float ব্যবহার করলে সঠিক ফলাফল পাওয়া যায়।
3. **মেমোরি ব্যবহারের দক্ষতা**: float ডেটা টাইপ সাধারণত ৪ বাইট মেমোরি নেয়, যা অনেক ছোট ডেটার জন্য যথেষ্ট।

### উদাহরণ:
যদি আপনি একটি আয়তক্ষেত্রের দৈর্ঘ্য এবং প্রস্থ দশমিক সংখ্যায় দিতে চান, তাহলে float ব্যবহার করা প্রয়োজন। 

যদি আপনি শুধু পূর্ণ সংখ্যা (যেমন 300 এবং 200) ব্যবহার করেন, তাহলে **int** ডেটা টাইপও ব্যবহার করা যেতে পারে। তবে, float ব্যবহারে ভবিষ্যতে দশমিক মান যোগ করার ক্ষেত্রে সুবিধা থাকবে।
